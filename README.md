# Su Tüketim Davranış Analiz Dashboard

Bu proje, su tüketim verilerini analiz eden bir Streamlit dashboard uygulamasıdır.

## Özellikler
- Tüketim dağılımı görselleştirmesi
- Zone bazlı analiz
- Detaylı filtreleme seçenekleri
- İleri analiz grafikleri

## Kurulum
```bash
pip install -r requirements.txt
streamlit run main.py
